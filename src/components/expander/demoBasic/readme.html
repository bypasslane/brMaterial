Expander

Expanders can hold content and show them when the expander is opened, you can set he height of the expander or it with automatically set the height to the contents height

Expanders can be controlled in 3 ways. You can add a <br-expander-header> that will open the expander when clicked.
You can add a [br-open] attribute that takes a boolean.
you can use the $brExpander service to controll them by their [br-componenet-id] name.
